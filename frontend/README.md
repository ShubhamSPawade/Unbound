Unbound Frontend

Overview

Frontend for Unbound built with Next.js 16, following a neobrutalism design approach focused on bold visuals and simplicity.

Tech Stack

- Next.js 16
- React
- Tailwind CSS

Setup

git clone https://github.com/ShubhamSPawade/Unbound.git
cd Unbound/frontend
npm install
npm run dev

Features

- Minimal, bold UI (neobrutalism)
- Fast and optimized rendering
- Scalable component structure

Env

NEXT_PUBLIC_API_URL=your_backend_url
